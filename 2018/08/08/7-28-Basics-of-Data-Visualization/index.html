7.28- Basics of Data Visualization

讲师简介:
陈为，浙江大学CAD＆CG国家重点实验室教授，主要研究方向为计算机图形学和可视化。在过去五年中，他发表了60多篇期刊和会议论文。他目前的研究兴趣包括科学可视化，可视化分析和生物医学图像计算。
课程简介：
课程从什么是可视化、可视化如何发展、如何可视化三个问题出发，结合大量实例，介绍了可视化理论和技术基础。
数据可视化的处理模型：1999年Card等人提出了信息可视化的流水线处理模型，而后循环模型等多种可视化模型陆续出现。这些模型抽象了从原始数据经数据处理、可视映射，最终形成数据可视的表达的过程。
可视编码的原则：针对不同的数据类型和数据模式，应选择不同的视觉通道进行表达。此外，颜色、形状等视觉通道具有的心理特性和视觉特性，也决定了应当按照一定原则设计视觉映射。本部分通过比较不同方法下的视觉效果详细介绍了准则。
可视分析的模型：以嵌套模型为例，针对主要的数据类型—图表、网络、空间数据等，通过大量实例说明了这一模型。
此外，本日课程同样介绍了可视化领域新的热点问题。在下午的课程中，由浙江大学CAD&CG实验室的成员介绍了D3.js和PowerBI这两种可视化工具的特性和用法。
